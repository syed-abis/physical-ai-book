# Quickstart: Console Todo App

**Date**: 2025-12-06

This guide provides instructions on how to run the console to-do application.

## Prerequisites
- Python 3.13+
- UV

## Running the Application
1.  Navigate to the project's root directory.
2.  Install dependencies: `uv pip install -r requirements.txt` (Note: `requirements.txt` will be created during implementation).
3.  Run the application: `python src/todo/main.py`
